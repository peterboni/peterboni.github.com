PhotoArchive to Your Mac

Follow these steps to setup your Mac & PhotoArchive, so that you can easily PhotoArchive to your Mac Pictures folder.

These instructions are for Mac OS X Mountain Lion, Version 10.9.2, and PhotoArchive 1.0.

1. Launch Terminal

Use either of the following ways to launch Terminal.

  1. Open Finder
  2. Go to Applications - Utilities
  3. Open Terminal

or

  1. Open Spotlight
  2. Type in ‘Terminal’ (without the quotes)
  3. Open the ‘Top Hit’ (it should be Terminal)

Take note of your Mac User Account Name.

2. Start the Mac OS X FTP server

Type in (or copy & paste) the following command and press enter.

sudo -s launchctl load -w /System/Library/LaunchDaemons/ftp.plist

You may be prompted for an administrator username and/or password. Enter them if prompted.

3. Test FTP server & Get IP address

The next step is to test the Mac OS X FTP Server.

  1. Type ‘ftp localhost’ and press enter.
  2. Press enter (it will default to your Mac User Account Name).
  3. Enter your password and press enter. This is the password you enter to login to your Mac.

The Terminal window should look like this.

Now check you can access the Pictures folder.

  1. Type ‘cd Pictures’ and press enter.
  2. Type ‘ls’ and press enter.

You should see the contents of your Pictures folder.

Now to get the IP address of your Mac on the network.

  1. Open System Preferences
  2. Click ‘Network’
  3. Click on either ‘Ethernet’ or ‘Wi-Fi’ (in that order & whichever one is green)
  4. Take note of the numeric IP address.

4. Install & Launch PhotoArchive on your iPhone

5. Configure PhotoArchive Settings

  1. Tap Settings
  2. Tap FTP
  3. Tap Host (x2), enter your Mac IP address (from Step 3 above), tap Save
  4. Tap Name (x2), enter your Mac User Account Name (from Step 1 above)
  5. Tap Password, enter your Mac User Password (the same one you entered in Step 3 above), tap Save
  6. Tap Folder, tap Pictures, tap ‘Create Folder’ & enter a name, tap OK, then Save
  7. Tap Done (x2)

At Step 8 above, your FTP view in PhotoArchive should look similar to this.
